### 🗃️ Project 1 — Student Enrolment Management System

**Type:** Group Project | **Role:** Database Designer  
**Technologies:** Java, MySQL  
**Repository:** [ADP-PROJECT](https://github.com/MatumaSophia/ADP-PROJECT.git)

#### 📋 Description
This system was developed as part of an Application Development module at CPUT. It provides a dual-interface platform where an **Admin** can log in, enrol new students, and manage subjects/modules. Students can then sign in using credentials created by the admin and self-enrol into available modules.

#### 🔍 My Contribution
As the **Database Designer**, I was responsible for:
- Designing the relational database schema to support admin and student roles
- Creating and managing MySQL tables for users, modules, and enrolments
- Ensuring data integrity through the use of primary keys, foreign keys, and constraints
- Collaborating with the team to integrate the database with the application's front end

#### 💡 Evidence of Learning
This project deepened my understanding of **relational database design** and how a well-structured database supports complex application logic. I learned how to model real-world relationships (students, subjects, enrolments) into database tables and how role-based access control works at the data level.

#### 🔗 Connect
This project directly connects to my career goal of becoming a backend or full-stack developer, where strong database skills are essential. It also reinforced my ability to work within a team, communicate technical decisions, and deliver a working component of a larger system.
